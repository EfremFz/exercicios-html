Introdução

O HTML ou HyperText Markup Language que significa "Linguagem de Marcação de Hipertexto" é a estrutura básica das páginas exibidas atualmente nos navegadores

Junto com algumas outras tecnologias como CSS e JavaScript forma a página que estamos acostumados a usar.

Bloco de Notas

Por se tratar de uma linguem de marcação de texto, não é necessário uma IDE para trabalhar com HTML, você pode usar qualquer editor de texto, basta no momento de salvar não se esquece de colocar a extensão no nome do arquivo EX: minhaPagina.html

A diferença entre trabalhar com HTML em uma IDE para o editor de texto são as funções de como a sugestão de código e o auto completar, facilidades que ajudam na produtividade.

Estrutura básica do HTML

Primeiro identificamos que o arquivo se trata de uma página HTML, para isso usamos “!DOCTYPE html”

O HTML é uma estrutura básica onde informações importantes do arquivo, como title que é nada mais que o nome que aparece na aba do navegador e outras informações como link da página de estilo. Essas informações ficam dentro da tag head

Depois vem o Body , dentro desta tag é onde realmente é editada a página que será visível para quem acessar

Por fim no final da página é fechada todas as tags abertas para criar a estrutura e sinalizar para o navegador que aquele é o fim do arquivo.

TAGS

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.
